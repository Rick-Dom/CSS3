Normalize e reset css

Os problemas surgem por conta dos navegadores,pois algumas propriedades css sao renderzadas de maneira difrente entre elas. E para resolver isso surgiu o reset css que nada mais é do que uma tecnica para padronizar a visualização das paginas.

De certa forma acabamos zerando as propriedades de modo em que elas serão renderizadas da mesma forma nos diversos navegarores, nós resetamos essas propriedades reescrendo-as.

O normalize tem a mesma função, mas podemos que disser que é mais moderno e faz ainda mais coisas, dando mais consistência e fazer com que essas propriedades se comportem de forma mais normalizada nos navegadores

Chamamos ele antes do nosso arquivo css que vamos estilizar e também podemos usar pelo cdn copiando e importando diretamente da web

Mas qual devo utilizar?

Se voce quiser apenas normalizar a renderização desse elementos utilize o normalize.

Mas se voce for criar um projeto, e sabe que vai ter muitas costumizações dos elementos e deseja ter controle sobre eles crie o seu proprio reset. Quando mais simples melhor!

Uma opção mais simples, por exemplo, seria:

*{margin: 0;padding: 0; border: 0;}
a{text-decoration: none;}
ul{list-style: none;}
